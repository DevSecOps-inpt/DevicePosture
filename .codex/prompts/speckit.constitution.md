# speckit.constitution (Local Wrapper)

Use this wrapper to draft high-level project principles only.

Constraints:
- Keep output at architecture/governance level.
- Do not create granular implementation task lists.
- Convert execution tasks into Beads items.

Prompt skeleton:
1. Define project principles (quality, security, testing, performance, UX consistency).
2. Define constraints and non-goals.
3. Define decision-making rules and tradeoff policy.
4. Define review and verification expectations at high level.
